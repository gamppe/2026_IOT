# ch06 Flask 예제 파일

각 파일은 독립적으로 실행할 수 있습니다.

```bash
source ~/myenv/bin/activate
cd ch06_examples
python 6-1.py
```

## 포함 파일

- hello_flask.py
- 6-1.py ~ 6-12.py
- camera.py
- cctvapp.py
- templates/hello.html
- templates/helloWithScript.html
- templates/getcalc.html
- templates/postcalc.html
- templates/cctv.html
- static/README.txt

## 주의

- 6-4.py는 `static/image.jpg` 파일을 직접 넣어야 테스트할 수 있습니다.
- CCTV 예제는 `opencv-python`과 라즈베리파이 카메라/웹캠 환경이 필요합니다.
